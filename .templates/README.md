# Templates

Starter files for common editing tasks. Copy, don't link — these are meant to be filled in.

- **`chapter-template.md`** — Skeleton for a new chapter file. Matches the standardized v0.2.0+ structure (Operational Blueprint → Strategic Priorities → Financial Checkpoint → Phases → End of Chapter Checklist). Copy to `NN-chapter-NN.md` in the repo root where `NN` is the part prefix (`01`/`02`/`03`/`04`).
- **`changelog-entry-template.md`** — Skeleton for a new version entry. Copy the contents to the top of `CHANGELOG.md`, above the most recent entry.

For the full authoring workflow (when to bump version, what to update in sync), see `CLAUDE.md` in the repo root.
